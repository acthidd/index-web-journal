# index-web-journal

## 项目简介

`index-web-journal` 是一个用于归档和发布多个独立 HTML 页面的仓库。这里不针对任何特定域名或网站，仅作为个人或小团队维护的网页集合，记录和分享各类静态页面内容。

## 目录说明

仓库根目录下各文件夹或文件均代表一个独立页面项目，结构如下：

```
index-web-journal/
├── page-1/          # 第一个页面项目
│   ├── index.html
│   └── assets/
├── page-2/          # 第二个页面项目
│   ├── index.html
│   └── assets/
└── README.md        # 本文件
```

每个页面项目均包含其完整的 HTML 及依赖资源，可直接通过浏览器打开或部署在任意静态托管服务上。

## 页面归档说明

- 所有页面均为独立静态 HTML 文件，不依赖后端服务。
- 页面内容可能随时间更新或移除，不保证长期可用。
- 归档页面仅供学习、参考或存档用途，请勿用于其他目的。
